Detailed Description

A JSON Web Token (JWT) implementation.

See Also
RFC 7519

This implementation supports the 'none' and 'HS256' algorithms. Header parameters other than 'alg' and 'typ' aren't parsed. To comply with RFC 7515, headers with 'crit' parameter are invalid. Currently, only the 'exp' standard claim is validated. Applications that need to validate other claims need to do this in their validation logic.
